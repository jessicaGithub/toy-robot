# Toy-Robot Test

Command line toy robot simulator

- [Usage](#usage)
- [Commands](#commands)

# Usage

```sh-session
$ npm install -g robocop
$ npm link
$ toy-robot COMMAND
running command...
$ toy-robot (-v|--version|version)
toy-robot/0.0.0 darwin-x64 node-v11.10.1
$ toy-robot --help [COMMAND]
USAGE
  $ toy-robot COMMAND
...
```

# Commands

- toy-robot PLACE X,Y,F
- toy-robot MOVE
- toy-robot LEFT
- toy-robot RIGHT
- toy-robot REPORT
