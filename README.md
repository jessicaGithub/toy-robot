robocop
=======

Command line toy robot simulator

* [Usage](#usage)
* [Commands](#commands)

# Usage
```sh-session
$ npm install -g robocop
$ npm link
$ robocop COMMAND
running command...
$ robocop (-v|--version|version)
robocop/0.0.0 darwin-x64 node-v11.10.1
$ robocop --help [COMMAND]
USAGE
  $ robocop COMMAND
...
```
