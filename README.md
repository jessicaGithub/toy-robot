# Toy-Robot Test

Command line toy robot simulator

- [Usage](#usage)
- [Commands](#commands)

# Usage

```sh-session
$ npm install
$ npm link
$ toy-robot hello

USAGE
  $ toy-robot COMMAND
...
```

# Commands

- toy-robot PLACE X,Y,F
- toy-robot MOVE
- toy-robot LEFT
- toy-robot RIGHT
- toy-robot REPORT
